Insecticide

Insecticide

Insecticide is a software defect tracking system. Commonly known as a bug tracker. Insectcide is built with ASP.NET, MVC, C# and SQL. A role-based security system was implemented to create permission-based roles. The roles of Admin , Project Manager, Developer and Submitter restrict what the user can see and do.

The system was built with C# and .Net in 3 weeks. The front end was developed with HTML, Javascript and Bootstrap. The backend utlizies MVC and C#. The database was built with SQL Server and hosted on Azure. Please visit the site and try it out.
